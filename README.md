# 📈 NIFTY 50 | 5 Years Market Analysis Dashboard (2021-2025)

## Project Overview

This project is an end-to-end Power BI dashboard developed to analyze 5 years of NIFTY 50 market data (2021-2025).

The dashboard provides insights into market trends, price movements, volatility, turnover, and trading patterns through interactive visualizations.

## Tools Used

* Power BI
* Power Query
* DAX
* Data Cleaning
* Data Modeling
* Data Visualization

## Dataset

Source: NIFTY 50 historical market data (2021-2025)

Files used:

* 2021 dataset
* 2022 dataset
* 2023 dataset
* 2024 dataset
* 2025 dataset

## Data Cleaning Process

* Removed duplicate records
* Handled missing values
* Corrected data types
* Standardized column names
* Validated outliers

## DAX Measures Created

* Average Close Price
* Average Volatility
* Count of Shares Traded
* Maximum Close Price
* Maximum High Price
* Minimum Low Price
* Sum of Turnover

## Dashboard Insights

* Average close price showed steady growth over 5 years.
* Market volatility was highest in February and lowest in July.
* Turnover peaked in 2025.
* Share trading displayed seasonal patterns.
